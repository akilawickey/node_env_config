# node_prod_server
App with different env configs

#Steps
git clone https://github.com/akilawickey/node_prod_server.git

cd node_prod_server

To run this first create .env in the folder

inside .env file add your secret details

eg -  

DB_HOST=localhost
DB_USER=root
DB_PASS=s1mpl3

Then save it and 

npm install

npm start
